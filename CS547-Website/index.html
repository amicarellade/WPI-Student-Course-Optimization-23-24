Introduction

Our team set out to build a course planning tool for WPI students that will aid in their selection of classes to fulfill degree requirements. It differs from the available WPI planner tool in that it also includes the course ratings and student rankings for the course. Our goal with this project was to help students create a course schedule that will both fulfill their major requirements as well as understand the difficulty/ time needed for the courses available. Instead of having to search the course on a different site to see the ratings, all the information will be available to them in one place. We understand that course planning can be stressful for students and we are looking to create a way to make this process as simple as possible.

Data

OSCAR

Online Students Course Reports (OSCAR) is an interactive Tableau dashboard that aggregates student feedback on courses and professors to provide valuable insights for academic planning. The data includes metrics such as the average time students spend on courses, their predicted grades, and evaluations of professor quality. By visualizing these trends, OSCAR empowers students and faculty to make informed decisions about course selection and instructional improvement.

Workday WPI

Workday for universities is a cloud-based platform designed to streamline and integrate administrative and academic processes in higher education. It provides tools for managing student information, academic records, admissions, financial aid, payroll, human resources, and budgeting. By using Workday we would be able to extract course information for all courses in the 2023-2024 academic year.

RateMyProfessor

Rate My Professor (RMP) is an online platform that enables students to read and write reviews of professors from academic institutions in the United States, Canada, and the United Kingdom. Established as a peer-driven resource, RMP is designed to help students make informed decisions about their course selections and instructors. The website features ratings and feedback on various aspects of a professor's teaching style and classroom environment, including clarity, engagement, workload, grading fairness, and overall helpfulness. Additionally, many reviews include written comments, which often provide context or personal experiences from past students, giving a deeper understanding of what to expect from a specific professor or course.

Data Retrieval

For each data source, a new method was used to scrape and obtain the data necessary for our project to work. In this section we will go into detail to how data from each source was retrieved.

OSCAR

OSCAR exists as a Tableu dashboard, therefore, when using inspect element, the webpage does not consider dropdowns, text, bars, or numbers as elements on the page. With that being said, to retrieve the information we had to use an amalgamation of computer vision and web scraping techniques to first click, scroll, and take a screenshot. Once the screenshot was taken, X library was used to analyze and investigate the ordered pixels and what they truly represented.

Workday

This Python script uses Selenium to automate web scraping to extract course and professor information from a Workday page. After downloading and setting up the Chrome driver, the script navigates to the specified Workday URLs, prompting the user to manually log in and select certain filters within the page before continuing. Once logged in, the script extracts course names and professor names by locating HTML elements with specific XPATH values (promptOption for courses and compositeSubHeaderOne for professors). The text content of these elements is appended to respective lists. The script continuously scrolls down the page to load additional elements until no new elements are detected, indicating that all data has been loaded. Finally, the script saves the extracted data into separate CSV files for courses and professors and handles any exceptions, ensuring the browser is properly closed in all cases.
